// Even Number Filter with Java Streams

A simple Java program that filters even numbers from a list using lambda expressions and the Streams API.

// Features
- Filters even numbers using 'Stream.filter()' and lambda
- Handles edge cases: empty lists, all odd numbers
- Clean, readable functional-style code

// Usage
1. Clone the repository
2. Compile: `javac *.java`
3. Run: `java Main`

// Example Output   
Original List: [1, 2, 3, 4, 5] Even Numbers List: [2, 4] 

Original List: [] Even Numbers List: []

Original List: [1, 3, 5, 7] Even Numbers List: []

// Code Structure
- 'EvenNumberFilter.java': Contains the static 'filterEvenNumbers' method
- 'Main.java': Demonstrates filtering with multiple test cases

// Requirements
- Java 8 or higher
